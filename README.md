# 黑话翻译器

> 网络流行语 × 职场黑话 × 英文缩写 → 一看就懂的大白话

## 访问地址

**https://wsnidsw.github.io/hacker-translator/**

电脑、手机浏览器都能打开，24小时在线，无需安装。

## 功能

- **整段翻译**：粘贴含黑话的文字，自动识别并高亮，下方列出对照表
- **单词查询**：输入单个词查释义，支持模糊搜索
- **330条词库**：覆盖拼音缩写、谐音梗、网络流行语、职场黑话、游戏术语等

快捷键：输入框内 `Ctrl + Enter` 快速翻译

## 本地使用

双击 `index.html` 即可在浏览器打开，离线也能用。

## 更新词库

1. 编辑 `index.html` 文件，找到 `DICT` 部分
2. 按格式添加新词：`["黑话", "大白话解释"],`
3. 提交推送：
```bash
git add index.html
git commit -m "新增词条: xxx"
git push
```
推送后约30秒自动部署更新。
